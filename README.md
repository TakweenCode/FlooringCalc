# 🧮 حاسبة تكلفة الأرضيات (Flooring Cost Calculator)

تطبيق ويب تفاعلي وسهل الاستخدام مصمم لمساعدة الأشخاص والمقاولين في حساب التكلفة التقديرية لتركيب الأرضيات (سيراميك، باركيه، رخام، إلخ) للغرف والمساحات المختلفة بسرعة ودقة.

## ✨ المميزات

* **حساب مساحات متعددة:** إمكانية إضافة عدد غير محدود من الغرف وحساب مساحتها الإجمالية.
* **نسبة هالك مرنة:** إمكانية تعديل نسبة الهالك (القصات) حسب نوع المادة (الافتراضي 10%).
* **حساب التكاليف الشاملة:** إدخال سعر المواد، سعر التركيب (أجرة اليد)، ورسوم التوصيل لحساب التكلفة النهائية بدقة.
* **دعم العملة المحلية:** عرض جميع النتائج والتكاليف بالريال السعودي (ر.س).
* **نسخ النتائج:** زر لنسخ تقرير ملخص بالنتائج والتكاليف لمشاركته بسهولة عبر الواتساب أو غيره.
* **إعادة ضبط سريعة:** زر لتصفير جميع الحقول والبدء بحساب جديد بضغطة واحدة.
* **الوضع الليلي والنهاري:** واجهة مستخدم مريحة للعين تدعم الوضعين الداكن والفاتح.
* **عداد الاستخدام:** عداد حي (Real-time) يعرض عدد مرات استخدام الحاسبة من قبل جميع المستخدمين.
* **حركات تفاعلية (Animations):** واجهة مستخدم سلسة وجذابة بفضل الحركات الانتقالية.

## 🛠️ التقنيات المستخدمة

* **إطار العمل:** React.js (مع TypeScript)
* **بناء الواجهات:** Tailwind CSS
* **الأيقونات:** Lucide React
* **الحركات (Animations):** Framer Motion
* **قاعدة البيانات (للعداد):** Firebase Firestore
* **أداة البناء:** Vite

## 🚀 طريقة التشغيل محلياً (Local Development)

1. قم باستنساخ المستودع (Clone the repository).
2. تأكد من تثبيت [Node.js](https://nodejs.org/) على جهازك.
3. افتح موجه الأوامر (Terminal) في مجلد المشروع ونفذ الأمر التالي لتثبيت الحزم:
   ```bash
   npm install
   ```
4. قم بإعداد مشروع Firebase الخاص بك، وأضف ملف `firebase-applet-config.json` أو قم بتحديث إعدادات Firebase في ملف `src/firebase.ts`.
5. لتشغيل خادم التطوير، نفذ الأمر:
   ```bash
   npm run dev
   ```
6. افتح الرابط الذي يظهر في موجه الأوامر (غالباً `http://localhost:3000`) في متصفحك.

## 📱 طريقة الاستخدام

1. أدخل أبعاد الغرفة (الطول والعرض بالمتر).
2. اضغط على "إضافة غرفة أخرى" إذا كان لديك مساحات إضافية.
3. أدخل سعر المتر المربع للمواد وسعر التركيب.
4. أضف رسوم التوصيل (إن وجدت) وحدد نسبة الهالك.
5. اضغط على زر **"احسب التكلفة"**.
6. يمكنك نسخ النتائج بالضغط على زر **"نسخ النتائج"**.
